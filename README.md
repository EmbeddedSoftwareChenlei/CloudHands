## CloudHands
``` 
利用DPDK捕获流量，流量一般从交换机镜像一个端口并连接到设备工作网口。从捕获的流量进行重组，统计等操作，并对TCP，UDP等为基础的上层应用协议进行还原
为进一步的安全分析提供必要的数据。 

```
